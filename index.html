Being able to communicate worldwide is very important nowadays. It is good that nowadays it is enough to learn only one language: English. Of course, the best way to learn it is to use the language and speak it easily. But this is only possible with a certain vocabulary. You usually learn this vocabulary in school. But what if you don't? For that we want to introduce you this tool! Our idea is to provide a simple vocabulary trainer online, which can be used for free by anyone who knows the German language and wants to learn English. No registration is necessary, just get started right away!
